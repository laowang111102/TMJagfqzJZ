## 功能模块

### 新闻分类模块

- 用户输入新闻标题和内容，系统将自动进行分类。
- 分类结果展示，包含文本框显示，便于用户输入和查看分类结果。

### 新闻管理

- 以列表形式展示已分类的新闻信息及其分类结果。
